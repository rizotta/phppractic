Реализовано:
Создание онлайн-чата по технологии CRUD (create, read, update, delete).
Работа с происходит с БД MySQL, все комментари хранятся в ней.
Используются сессии для запоминания имён пользователей и возможности изменять/удалять только свои комментарии в пределах сессии.
Используется защита текстового поля от специальных символов.

Файл "SQL query to create DB.txt" - содержит SQL-запрос создания БД для работы с проектом.
